# Análise do Repasse do Preço do Petróleo para a Gasolina no Brasil

## Autor: Guilherme Reikdal Medeiros

## Objetivo

Este projeto tem como objetivo analisar se as variações no preço do petróleo internacional (Brent) são repassadas ao preço da gasolina no Brasil.

As principais questões investigadas são:

- O aumento do petróleo é repassado ao consumidor?
- A queda do petróleo reduz o preço da gasolina?
- Existe defasagem temporal no repasse?
- O repasse é proporcional?



## Dados Utilizados

- Preço da gasolina (Brasil) – ANP
- Preço do petróleo (Brent) – mercado internacional
- Taxa de câmbio (USD/BRL) – Banco Central do Brasil
